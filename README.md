# Urbit String

String processing library for Urbit.

Work in progress, feedback and pull requests welcome!

### Installing / Using

1. clone this repo
1. cp -r ./* pier/desk/
1. From the dojo:
  * |start %string-tester
  * :string-tester %test

You will then see a series of tests and then a final status...

```
>=
%sum
%tail
... more %test_names ...
[%all-passed "(:"]
```
